# Hairstyle Analysis Tools

A utility package for [Hairstyle Analysis](https://www.hairstyleanalysis.org/).

## Installation

```bash
go get github.com/susanzhang003/hairstyle-analysis
```

```bash
composer require susanzhang003/hairstyle-analysis
```

## About

This package provides helper utilities for working with the Hairstyle Analysis platform - an AI-powered hairstyle analysis tool.
